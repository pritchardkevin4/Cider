# Cider BAC Calculator Support

Welcome to the official support page for Cider BAC Calculator, an iOS app designed to help you track your blood alcohol content with precision.

## Frequently Asked Questions

### How accurate is the BAC calculation?
Cider uses the Widmark formula, which is the standard method for estimating blood alcohol content. However, this is an estimate only and should never be used to determine if you're sober enough to drive.
